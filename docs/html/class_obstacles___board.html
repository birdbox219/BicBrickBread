Obstacles_Board::Obstacles_Board ( )

Constructs an empty board with no moves made.

Creates an empty 6×6 board, initializes the empty cell symbol, and generates all 4-in-a-row win masks if they haven't been generated already.

Initializes bitboards, empty cell character, and loads the static win masks if needed.

Board layout uses 36 bits (0..35), mapping: index = row * 6 + col

Bitboards used:

  • boardX bits for X moves
  • boardO bits for O moves
  • boardTraps bits for traps (blocked cells)

Attempts to place symbol s at (r, c).
