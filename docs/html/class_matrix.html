Detailed Description

template<class T>
class Matrix< T >

Lightweight 2D matrix with contiguous row-major storage.

Designed primarily for neural-network operations:

  • Efficient indexing via a flattened vector
  • Minimal dynamic allocations
  • Element-wise ops, Hadamard ops, transpose, matrix multiply

The class DOES NOT perform implicit shape broadcasting. All operations validate dimensional consistency.
