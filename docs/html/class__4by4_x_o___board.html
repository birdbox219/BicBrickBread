overridevirtual

Applies a sliding move.

Attempts to move a token from its source to destination.

Parameters
movePointer to _4by4XO_Move
Returns
True if move is valid and applied

Rules enforced:

  • Move must be within board bounds.
  • Source must contain the correct player's symbol.
  • Destination must be empty.
  • Move must be exactly one step in Manhattan distance.
