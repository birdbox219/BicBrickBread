Detailed Description

Game board implementation for the Word-based Tic-Tac-Toe variant.

This class specializes the generic Board<char> template to support:

  • Word formation and dictionary lookup
  • Forward and backward word detection
  • Last-player tracking
  • Terminal-state evaluation

Constructs and initializes the Word Tic-Tac-Toe board.

Constructs a new Word_XO_Board object.

Initializes:

  • Board dimensions
  • Empty cell marker
  • Move counter

  • Word dictionary needed for validation

    Initializes a 3x3 board with empty cells and loads the dictionary from "dic.txt" if not already loaded. Also stores reversed words for AI use. Throws a runtime exception if the dictionary file is missing.
