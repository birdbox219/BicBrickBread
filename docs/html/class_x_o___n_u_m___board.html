Detailed Description

Numerical Tic-Tac-Toe board.

Players place numbers instead of X/O:

  • Player 1 uses odd numbers {1,3,5,7,9}
  • Player 2 uses even numbers {2,4,6,8} The objective is to place numbers such that a row, column, or diagonal sums to 15.
