Detailed Description

Fully-connected (dense) neural network layer.

Provides:

  • Forward propagation
  • Backward propagation
  • Weight & bias storage

Activation functions and their derivatives are injected via std::function to follow Dependency Inversion Principle (DIP) and maintain encapsulation.
