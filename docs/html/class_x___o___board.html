Detailed Description

Represents a Tic-Tac-Toe (X-O) board.

This class inherits from the generic Board<char> and implements the specific logic for a 3x3 Tic-Tac-Toe game.

Responsibilities:

  • Maintain board state.
  • Apply player moves.
  • Check win, draw, and game-over conditions.
