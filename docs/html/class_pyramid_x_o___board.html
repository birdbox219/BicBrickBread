Detailed Description

Represents the game board for the Pyramid XO variant.

Only certain cells are playable, forming a pyramid shape: Row 0: (0,2) Row 1: (1,1)(1,2)(1,3) Row 2: (2,0)(2,1)(2,2)(2,3)(2,4)
