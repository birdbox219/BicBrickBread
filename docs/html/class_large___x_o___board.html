Detailed Description

5x5 Tic-Tac-Toe board implementation using a bitboard for fast evaluation.

The board tracks positions using three 32-bit integers:

  • boardX : bitmask for X
  • boardO : bitmask for O
  • boardXO : bitmask for all occupied cells (X | O)

This representation dramatically accelerates win detection and evaluation. The class also stores all possible 3-in-a-row masks for scoring.
